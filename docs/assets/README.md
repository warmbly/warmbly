# Image assets for the project README

These files populate the README at the repo root. Keep them small so the README
loads quickly on GitHub.

| File                        | Where it shows up                    |
|-----------------------------|--------------------------------------|
| `logo-dark.svg`             | Logo + wordmark, white, for dark GitHub theme |
| `logo-light.svg`            | Logo + wordmark, dark ink, for light GitHub theme |
| `dashboard-campaigns.png`   | Features: campaigns list              |
| `dashboard-inbox.png`       | Features: unified inbox               |
| `dashboard-mailboxes.png`   | Spare product screenshot (mailboxes)  |

The dashboard images are exports of the product design. When real, anonymized
screenshots of the running app are available, swap them in at the same paths and
sizes (1440 x 900) and the README needs no changes.
